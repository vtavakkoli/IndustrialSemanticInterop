# Scenario Catalog

- `simple_sensor.yaml`: scalar sensor flow.
- `metadata_rich.yaml`: metadata-rich TEDS-like payload.
- `semantic_normalization.yaml`: explicit unit conversion.
- `multifield_telemetry.yaml`: OPC UA node-write bridge path.
- `secure_flow.yaml`: representative authenticated hybrid path.

Each scenario specifies source/target protocol, payload schema, mapping rules, expected output, validation criteria, and benchmark parameters.

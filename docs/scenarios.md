# Scenario Model

The benchmark supports base and complex/fault scenarios through configurable flags:

- `missing_metadata`
- `ambiguous_mapping`
- `unit_mismatch`
- `ontology_service_down`
- `opcua_endpoint_down`
- `authentication_failure`
- `high_load`
- `secure_and_semantic`
- `multi_constraint_mixed`

Scenarios are generated reproducibly from config combinations of scale, security mode, and scenario flags.
